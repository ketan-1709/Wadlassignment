The year blocks below provide a simple static presentation of a typical B.E. computer-related learning path at PICT, organized into FE, SE, TE and BE for mobile viewing.

FE - First Year Engineering

Engineering Mathematics

Builds the mathematical foundation for problem solving, calculus, matrices, and analytical thinking.

Engineering Physics

Introduces physical principles useful in electronics, waves, materials, and modern technology.

Basic Electrical and Electronics Engineering

Covers circuit basics, electrical systems, and introductory electronic components.

Programming Fundamentals

Introduces logic building, algorithms, and coding basics for engineering students.

SE - Second Year Engineering

Data Structures

Teaches arrays, linked lists, stacks, queues, trees, and efficient data organization methods.

Object Oriented Programming

Focuses on classes, objects, inheritance, polymorphism, and modular software design.

Digital Logic Design

Explains number systems, logic gates, combinational circuits, and sequential circuits.

Discrete Mathematics

Provides the basis for graph theory, logic, sets, relations, and computer science reasoning.

TE - Third Year Engineering

Database Management Systems

Explains relational databases, SQL, normalization, and efficient data storage concepts.

Operating Systems

Covers processes, memory management, scheduling, and system-level resource handling.

Computer Networks

Introduces communication models, protocols, routing, and internet architecture.

Software Engineering

Focuses on requirement analysis, software design, testing, and project development practices.

BE - Final Year Engineering

Machine Learning / Elective Subjects

Allows students to explore specialized domains such as AI, data science, security, or cloud computing.

Major Project

Gives practical experience in designing, developing, and presenting a complete engineering solution.

Seminar

Improves technical presentation, research reading, and communication skills.

Internship / Industry Exposure

Connects academic learning with real-world engineering applications and workplace practice.
